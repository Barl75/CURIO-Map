Project introduction

This map gallery is fantastic.

DATA SOURCE

All layers in CURIO map gallery are using public data source. Most data are published by City of Columbus, Mid-Ohio Regional Planning Commission (MORPC) and Center for Urban and Regional Analysis (CURA).

LAYER CATEGORIZATION

Based on layers' nature, all layers are divided into Transportation (with orange edge), Environment (with green edge) and Social (with blue edge).

COMPATIBILITY

CURIO map gallery support most mainstream browsers (Chrome, Firefox, safari, etc.) on Windows, MacOS and Android pad.

BASEMAPS

Basemaps are provided by Esri and hosted on ArcGIS online. Basemaps can be integrated into webmaps for free.

MAP TEMPLATE

This map layout is adapted by Luyu Liu based on a bootleaf template created by Brian McBride
